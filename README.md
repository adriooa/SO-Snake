# SO-Snake
### Snake usando threads para disciplina de Sistemas Operacionais

<img src="res/snake.png" width="350">

## Instruções

- Use o Makefile para te axuliar. 
  - Use ``` make ``` para instruções.
- Instalação:
  - Rode ``` . ./script.sh ``` e ``` make compile ```
- Jogar:
  - Rode ``` make run ```

## Requisitos:

- G++11
